There are presently at least six server groups located around the world for playing World of Tanks. Each server group has its own Official Web Site. Five of the server groups are operated by Wargaming.net, while a sixth is operated by KongZhong in China. Each server runs independently, so bonuses, promotions, and time of updates are different. Progress in accounts in one server cannot be moved to another server unless in special occasions. However, they do get the same content in updates usually. It is usually best to choose a server closest to you so that your ping-rates will be low and you can be more effective in the tank of your choice.

Miscellaneous Information

  • Game rules allow 1 account per Server
  • All normal maintenance takes about 30 to 45 minutes to complete
